Place custom webfonts here.

Expected paths for Neue Haas Display (Medium):
- /public/fonts/NeueHaasDisplay-Medium.woff2
- /public/fonts/NeueHaasDisplay-Medium.woff

If your filenames differ, update @font-face in src/app/globals.css accordingly:
@font-face {
  font-family: 'Neue Haas Display';
  src: url('/fonts/YourFile.woff2') format('woff2');
  font-weight: 500;
}
